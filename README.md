Gulpstart
=============

Quickly start a project with Gulp.js

### Features

- JS uglify
- SASS, LESS & Stylus pre-processor
- JS hint
- JS minify
- Image minify
- CSS, JS injection 
- Clean files

### Usage
It's assumed you have `npm` installed and the `gulp-cli`.

1. clone the repo

		git clone https://github.com/one-aalam/gulpstart.git

2. install dependencies

		npm install

3. start gulp

		gulp
